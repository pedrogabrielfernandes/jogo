## 👨‍💻 Projeto

Este repositório reúne os arquivos, códigos e materiais desenvolvidos para o projeto de criação de um jogo, proposto na disciplina de Algoritmos e Estrutura de Dados I.

O projeto tem como finalidade aplicar, na prática, conceitos fundamentais da programação, lógica computacional e organização de dados, utilizando a construção de um jogo como ferramenta de aprendizado.

## 🚀 Objetivo

Este projeto tem como objetivo desenvolver um jogo aplicando os conhecimentos adquiridos na disciplina de Algoritmos e Estrutura de Dados I, estimulando o raciocínio lógico, a resolução de problemas e a prática de programação.

Além disso, o trabalho busca incentivar o desenvolvimento em equipe, a organização do código e a utilização de boas práticas durante a construção do projeto.
